 📊 Data Visualization using Python (Matplotlib)

📌 Description
This project demonstrates different types of data visualization using Python and Matplotlib library. 
It includes various charts like line plot, histogram, pie chart, scatter plot, and subplots.

🚀 Features
- Line Chart with markers and styling
- Multiple line comparison (Year-wise sales)
- Histogram for data distribution
- Pie Chart for percentage visualization
- Scatter Plot for relationship analysis
- Subplots (Multiple charts in one figure)
- 
🛠️ Technologies Used
- Python
- Matplotlib
- Random Module
  
 📥 Installation
Install required library using:
pip install matplotlib

 ▶️ Usage
Run the Jupyter Notebook or Python file:
jupyter notebook

## 📊 Charts Included
- Line Plot
- Styled Line Plot
- Multiple Line Plot
- Histogram
- Pie Chart
- Scatter Plot
- Subplots (Bar + Scatter)

## 📈 Output
Displays different types of graphs for better understanding of data.
